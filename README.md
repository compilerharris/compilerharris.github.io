## GitHub Pages Portfolio (Static)

This portfolio is **100% static** and GitHub Pages compatible:
- `index.html`
- `style.css`
- `script.js`
- `assets/`

### Add your CV PDF

Put your PDF at:
- `assets/Haris_Shaikh_CV.pdf`

The "Download CV" button links to that filename.

### Deploy (GitHub Pages)

- Push this folder to a GitHub repository.
- In GitHub: **Settings → Pages**
  - **Build and deployment**: Deploy from a branch
  - Branch: `main` (or `master`) and `/ (root)`
