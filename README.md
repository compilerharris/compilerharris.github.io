A modern, interactive personal portfolio built using **HTML, Tailwind CSS, and JavaScript**, featuring smooth animations and a premium UI experience.

## ✨ Features

* ⚡ Modern UI with smooth animations (Framer-like experience using GSAP / CSS)
* 🎯 Fully responsive (mobile-first design)
* 🌙 Dark theme with elegant styling
* 🎬 Scroll-based animations and transitions
* 📊 Animated skills and experience sections
* 🧭 Sticky navigation with active section tracking
* 📈 Scroll progress indicator
* 💡 Clean and modular code structure

## 🚀 Getting Started


### 2. Open Locally

---

## 🌍 Deployment (GitHub Pages)

1. Push code to GitHub
2. Go to **Settings → Pages**
3. Select:

   * Branch: `main`
   * Folder: `/root`
4. Your site will be live at:

   ```
   https://compilerharris.github.io/
   ```
